# Time Series in Java

An implementation of common time series calculations in Java.

### Current calculations include the following:
- [Mean](https://github.com/navdeep-G/timeseries-java/blob/master/src/main/java/collect/TSCollect.java)
- [Variance](https://github.com/navdeep-G/timeseries-java/blob/master/src/main/java/collect/TSCollect.java)
- [Standard Deviation](https://github.com/navdeep-G/timeseries-java/blob/master/src/main/java/collect/TSCollect.java)
- [Minimum](https://github.com/navdeep-G/timeseries-java/blob/master/src/main/java/collect/TSCollect.java)
- [Maximum](https://github.com/navdeep-G/timeseries-java/blob/master/src/main/java/collect/TSCollect.java)
- [Autocovariance](https://github.com/navdeep-G/timeseries-java/blob/master/src/main/java/collect/TSCollect.java) 
- [Autocorrelation](https://github.com/navdeep-G/timeseries-java/blob/master/src/main/java/collect/TSCollect.java)
- [Partial Autocorrelation](https://github.com/navdeep-G/timeseries-java/blob/master/src/main/java/collect/TSCollect.java)

### Current algorithms include the following:
